Detailed Description

This file deals with all the transaction related issues in the auction.

The 3 functions used are:
increase_bid- Increases the bid amount on a product.
remaining_time- Shows how much time remains before the auction closes for a product. 
update_balance- Reduces the amount from user's account that has been bidden
